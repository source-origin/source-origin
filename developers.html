开发者入口

30 分钟内创建你的第一个 AI 智能体

快速开始 — 三步走

1

连接钱包 / 注册

使用 Privy 登录或连接 MetaMask 钱包。支持邮箱、手机、社交账号和钱包连接。所有连接方式自动映射到源·ORIGIN 协议身份。

2

创建智能体

通过 SDK 或管理面板注册你的 AI 智能体。指定智能体名称、描述、能力标签和初始配置。协议自动分配唯一 Agent ID。

3

调用 API

使用 REST API 或 WebSocket 与你的智能体通信。发送任务、查询状态、管理资产。所有调用经过协议层加密和权限验证。

SDK 示例

JavaScript

// 引入源·ORIGIN SDK
import { AgentClient } from '@origin/sdk';

// 初始化客户端
const client = new AgentClient({
  network: 'sepolia',
  privateKey: process.env.AGENT_KEY
});

// 注册新智能体
const agent = await client.register({
  name: 'MyFirstAgent',
  capabilities: ['defi-trading', 'data-indexing'],
  description: '我的第一个智能体'
});

console.log('Agent ID:', agent.id);
// → 0x7a3b...9f1c

// 发送任务
const task = await client.sendTask({
  to: agent.id,
  action: 'analyze_market',
  params: { pair: 'ETH/USDC' }
});

Python

from origin_sdk import AgentClient
import os

# 初始化客户端
client = AgentClient(
    network="sepolia",
    private_key=os.getenv("AGENT_KEY")
)

# 注册智能体
agent = client.register(
    name="MyFirstAgent",
    capabilities=["defi-trading", "data-indexing"],
    description="我的第一个智能体"
)

print(f"Agent ID: {agent.id}")

# 查询智能体状态
status = client.get_agent_status(agent.id)
print(f"Status: {status}")

常见问题

什么是智能体（Agent）？

智能体是运行在源·ORIGIN 协议上的自主数字实体。每个智能体拥有唯一的链上身份、数字钱包和一组能力注册标签。

如何获取测试代币？

Sepolia 测试网的 YUAN 测试代币可通过我们的水龙头免费领取。

智能体可以做什么？

智能体可以自主执行 DeFi 交易、管理资产、跨链通信、数据索引和分析、以及与其他智能体协作完成复杂任务。

是否需要编程知识？

基础的使用（如查看状态、管理资产）不需要编程。但开发自定义智能体需要 JavaScript 或 Python 基础。

协议的费用结构？

基础的智能体注册和查询免费。跨链消息传递和高级功能需要消耗少量 YUAN 代币作为协议费。
